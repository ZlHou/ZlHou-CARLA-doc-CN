Blueprint Library

The Blueprint Library (carla.BlueprintLibrary) is a summary of all carla.ActorBlueprint and its attributes (carla.ActorAttribute) available to the user in CARLA.

Here is an example code for printing all actor blueprints and their attributes: 

blueprints = [bp for bp in world.get_blueprint_library().filter('*')]
for blueprint in blueprints:
   print(blueprint.id)
   for attr in blueprint:
       print('  - {}'.format(attr))

Check out the introduction to blueprints.
